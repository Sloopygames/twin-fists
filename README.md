# Twin Fists

Twin Fists drops you into a neon back-alley rumble where you chain punches, kicks, jump attacks, thrown junk, and emotion-fueled specials to survive escalating enemy waves.

Part of the [sloopygames.com](https://sloopygames.com) collection.

## Play

Open `index.html` in any modern browser. No build step required, but the game loads Google Fonts and Tone.js from CDNs.

## How to Play

- Clear each enemy wave to advance and keep scoring damage as the alley gets more dangerous.
- Survive as long as possible with your health intact; the run ends when you are knocked out.
- Punch near loose props to pick them up or throw them, punch stunned enemies to grab and launch them, and spend a full special meter with punch + kick for a mood-based super move.
- Expect elite waves every 5 rounds, with tougher enemy archetypes and faster pressure as the run continues.

## Controls

- **Arrow keys / WASD** -- Move
- **J** -- Punch / pick up props / grab or throw stunned enemies
- **K** -- Kick / drop held props
- **Space** -- Jump
- **J + K** -- Use special when the meter is full
- **Touch D-pad** -- Move on mobile
- **Touch Punch / Kick / Jump buttons** -- Attack and jump on mobile

## Features

- Wave-based street brawler action with score chasing and elite rounds every five waves.
- Punch, kick, and jump-kick combat with grabs, throws, and breakable alley props.
- Mood-driven special abilities that change across the run, from healing and dashes to area attacks and damage buffs.
- HTML5 Canvas presentation with reactive music, sound effects, and mobile touch controls.

## Tech

- HTML, CSS, vanilla JavaScript
- HTML5 Canvas
- Tone.js via CDN for music and sound effects
- Google Fonts via CDN
